# Hello Vuepress

#### 执行自动化执行命令
```bash
npm run deploy '***' // ***为输入的提交信息 macOs下的

sh deploy.sh 'xxx' // xxx为输入的提交信息 window下的，在git-bash下执行文件
```

#### 释放默认主题功能组件
```bash
vuepress eject docs
```
