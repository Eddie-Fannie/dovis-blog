# 239.滑动窗口最大值
| 题目类别 | 难度
| ---- | ---- |
| 堆/滑动窗口 | easy |

## 题目
> 给定一个数组 `nums` 和滑动窗口的大小 `k`，请找出所有滑动窗口里的最大值。

## 例子
```bash
输入： nums=[1,3,-1,-3,5,3,6,7] 和 k=3
输出：[3,3,5,5,6,7]
解释：
滑动窗口的位置               最大值
-------------              --------
[1 3 -1] -3 5 3 6 7          3
1 [3 -1 -3] 5 3 6 7          3
1 3 [-1 -3 5] 3 6 7          5
1 3 -1 [-3 5 3] 6 7          5
1 3 -1 -3 [5 3 6] 7          6
1 3 -1 -3 5 [3 6 7]          7
```

## 解法
1. 暴力法
> 移动滑动窗口，比较最大值

```js
/**
 * @param {number[]} nums
 * @param {number} k
 * @return {number[]}
 */
var maxSlidingWindow = function(nums, k) {
    if (k <= 1) return nums;
    const res = [];
    for (let i = 0; i < nums.length - k + 1; ++i) {
        res.push(Math.max(...nums.slice(i, i + k)));
    }
    return res;
};
```
